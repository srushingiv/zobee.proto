API Services Retrieve Teams: Fetches teams within the league
Route: team/allteams

Team Players: Fetches roster given teamId
team/userTeamPlayers/{teamId}

Free Agents: Given a game position, retrieves available free agents, available positions : ('QB','RB','WR', 'TE','K','D/St')
Route: roster/freeAgentsByPosition/{position}

Add Player : Adds a freeagent to a team , if status = 1 , player is added to the active roster, if status = 0 player is added to the active roster, spot ( 1-8 ) defines the spot within either active or roster
Route: team/addplayer/0/{playerid}/{status}/{teamId}/{spot}/0

Remove Player: Removes a player from team roster
Route: roster/removePlayer/{leagueId}/{userId}/{playerId}
